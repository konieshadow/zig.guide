# Hello World

创建一个名为 `main.zig` 的文件，包含以下内容：

```zig
const std = @import("std");

pub fn main() void {
    std.debug.print("Hello, {s}!\n", .{"World"});
}
```

使用 `zig run main.zig` 来构建并运行它。在此示例中，`Hello, World!`
将被输出到 stderr，并假定永远不会失败。

:::warning 出现 'invalid bytes'
确保您的 `main.zig` 文件使用 UTF-8 编码，因为 Zig 当前不支持其他的编码类型。要将文件重新编码为 UTF-8，请运行 `zig fmt main.zig` 然后在编辑器中重新打开文件。
:::
