# 欢迎

[Zig](https://ziglang.org) 是一种通用编程语言和工具链，用于维护**健壮**、**最优**和**可重用**的软件。

:::warning

Zig 的最新发布版本是 0.11.0 而它目前并不稳定。

:::

要遵循本指南，我们假设您已具有：

- 先前的编程经验
- 对底层编程概念的一些理解

了解 C、C++、Rust、Go、Pascal 或类似语言将有助于您遵循本指南。您必须拥有一个编辑器、终端和可用的网络。
