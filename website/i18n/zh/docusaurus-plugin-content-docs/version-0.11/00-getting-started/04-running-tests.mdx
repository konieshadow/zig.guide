# 运行测试

在本指南中，代码示例通常作为可运行的测试给出。在继续之前，请确保您可以成功运行它们。

### 成功

将以下文本保存为 `test_pas.zig`，并运行 `zig test test_pass.zig`；您应该在终端中看到 `All 1 tests  passed.`。

<CodeBlock language="zig">{Success}</CodeBlock>

:::note

本指南中一些代码示例的导入会在顶部隐藏，请确保通过点击代码块右上角的复制按钮来获取它们。

:::

### 失败

现在，将以下文本保存为 `test_fail.zig` 并观察它失败。

```zig
const std = @import("std");
const expect = std.testing.expect;

test "always fails" {
    try expect(false);
}
```

应该输出如下内容：

```
Test [1/1] test.always fails... FAIL (TestUnexpectedResult)
/usr/lib/zig/std/testing.zig:515:14: 0x2241ef in expect (test)
    if (!ok) return error.TestUnexpectedResult;
             ^
[...]/test_fail:5:5: 0x224305 in test.always fails (test)
    try expect(false);
    ^
0 passed; 0 skipped; 1 failed.
```
