# 安装

<Tabs
    defaultValue="linux"
    values={[

    <TabItem value="linux">
    考虑从您的发行版的包管理器获取(https://github.com/ziglang/zig/wiki/Install-Zig-from-a-Package-Manager) Zig。大多数主要的 Linux 发行版都包含最新的 Zig 版本。

    ### 手动安装

    1. [下载](https://ziglang.org/download/#release-0.11.0) Zig 的预构建版本。

        选择与您的 CPU 架构相匹配的 Zig 0.11 for linux 版本。如果您不确定正在使用哪种架构，可以通过以下方式找到：

        ```bash
        uname -m
        ```

    2. 使用 tar 提取存档，例如：

        ```bash
        tar xf zig-linux-x86_64-0.11.0.tar.xz
        ```

    3. 将 Zig 二进制文件的位置添加到您的 path 中，例如：

        ```bash
        echo 'export PATH="$HOME/zig-linux-x86_64-0.11.0:$PATH"' >> ~/.bashrc
        ```
    </TabItem>
    <TabItem value="windows">
    考虑从包管理器（例如 [chocolatey](https://chocolatey.org/)、[scoop](https://scoop.sh/) 或 [winget](https://learn.microsoft.com/en-us/windows/package-manager/winget/#install-winget)）获取 Zig。

    展示的所有命令都要在 powershell 内使用。

    ```powershell
    choco install zig
    ```
    ```
    winget install zig.zig
    ```
    ```
    scoop install zig
    ```

    ### 手动安装

    1. [下载](https://ziglang.org/download/#release-0.11.0) Zig 的预构建版本。

        选择适合您的 CPU 架构的 Zig 0.11 版本。大多数 Windows 系统使用 x86_64，也称为 AMD64。如果您不确定正在使用哪种架构，可以通过以下方式找到：

        ```powershell
        $Env:PROCESSOR_ARCHITECTURE
        ```

    2. 提取 Zig。

    3. 添加 Zig 到您的 path：

        <Tabs
        defaultValue="user"
        values={[
            {label: '当前用户', value: 'user'},
            {label: '系统', value: 'system'},
        ]}>
        <TabItem value="user">

        ```powershell
        [Environment]::SetEnvironmentVariable(
            "Path",
            [Environment]::GetEnvironmentVariable("Path", "User") + ";C:\_\zig-windows-_",
            "User"
        )
        ```
        </TabItem>

        <TabItem value="system">

        ```powershell
        [Environment]::SetEnvironmentVariable(
            "Path",
            [Environment]::GetEnvironmentVariable("Path", "Machine") + ";C:\_\zig-windows-_",
            "Machine"
        )
        ```
        </TabItem>
        </Tabs>

        关闭您的终端，并启动一个新终端。


    </TabItem>

    <TabItem value="macos">
    考虑从包管理器（例如 [brew](https://brew.sh/)）获取 Zig。

    ```
    brew install zig
    ```
    </TabItem>

</Tabs>

### 验证安装

使用 `zig version` 命令验证安装。输出应该看起来像：

```
$ zig version
0.11.0
```

### 附加功能

为了在编辑器中实现代码补全和转到定义，请考虑安装 [Zig 语言服务器](https://github.com/zigtools/zls/#installation)。

考虑加入一个 [Zig 社区](https://github.com/ziglang/zig/wiki/Community)。
