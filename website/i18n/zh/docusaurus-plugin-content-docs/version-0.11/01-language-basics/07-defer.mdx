# Defer

Defer 用于在退出当前块时执行语句。

<CodeBlock language="zig">{Defer}</CodeBlock>

当单个块中有多个 defer 时，它们会反向执行命令。

<CodeBlock language="zig">{DeferMulti}</CodeBlock>

Defer 对于确保资源在不再需要时被清理很有用。无需记住手动释放资源，您可以在分配该资源的语句旁边添加一个 defer 语句。
