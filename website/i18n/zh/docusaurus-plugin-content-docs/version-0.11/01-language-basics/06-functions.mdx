# 函数

**所有函数参数都是不可变的**——如果需要副本，用户必须显式地制作一个。与使用蛇形命名的变量不同，函数使用驼峰式。这是一个声明并调用简单函数的示例。

<CodeBlock language="zig">{Functions}</CodeBlock>

允许递归：

<CodeBlock language="zig">{FunctionsRecursion}</CodeBlock>

当递归发生时，编译器不再能够计算出最大堆栈大小，这可能会导致不安全行为——堆栈溢出。有关如何实现安全递归的详细信息将在以后介绍。

可以使用 `_` 而不是变量或常量声明来忽略值。这不适用于全局范围（即它仅适用于函数内部或者块），并且对于忽略从函数返回的值很有用（当你不需要它们时）。

```zig
_ = 10;
```
