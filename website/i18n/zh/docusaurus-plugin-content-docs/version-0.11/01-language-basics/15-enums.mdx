# 枚举

Zig 的枚举允许您使用一组有限的命名值来定义类型。

让我们声明一个枚举。

```zig
const Direction = enum { north, south, east, west };
```

枚举类型可能具有指定的（整数）类型。

```zig
const Value = enum(u2) { zero, one, two };
```

枚举的序数值从 0 开始。可以使用内置函数 [`@intFromEnum`](https://ziglang.org/documentation/master/#intFromEnum) 访问它们。

值可以被覆盖，下一个值从那里继续。

<CodeBlock language="zig">{EnumOrdinalOverride}</CodeBlock>

枚举可以被赋予方法。它们充当命名空间函数，可以使用点语法来调用。

<CodeBlock language="zig">{EnumMethods}</CodeBlock>

枚举也可以被赋予 `var` 和 `const` 声明。它们充当命名空间全局变量，它们的值与枚举类型的实列无关且不会附加到枚举上。
