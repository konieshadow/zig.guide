# 切片

切片可以被认为是一对 `[*]T`（指向数据的指针）和一个 `usize`（元素的数量）。它们的语法是 `[]T`，其中 `T` 是子类型。
当您需要操作任意数量的数据时，可以在 Zig 中大量地使用切片。
For 循环也可以对切片进行操作。Zig 中的字符串文字可转换为 `[]const u8`。

这里，语法 `x[n..m]` 用于从数组创建切片。这称为切片，并创建从 `x[n]` 开始到 x[m - 1]` 结束的元素切片。
此示例使用 const 切片，因为切片指向的值不允许被修改。

<CodeBlock language="zig">{Slices}</CodeBlock>

当这些 `n` 和 `m` 在编译期都已知时，切片实际上会生成一个指向数组的指针。这不是问题，因为指向数组的指针即 `*[N]T` 可转换为 `[]T`。

<CodeBlock language="zig">{Slices2}</CodeBlock>

当你想要切片到末尾时，也可以使用语法 `x[n..]`。

```zig
test "slices 3" {
    var array = [_]u8{ 1, 2, 3, 4, 5 };
    var slice = array[0..];
    _ = slice;
}
```

可以切片的类型有数组、许多指针和切片。
