# Switch

Zig 的 `switch` 既可以用作语句，也可以用作表达式。所有分支的类型必须都可以转换为 switch 表达式的值的类型。所有可能的值必须有一个对应的分支——值不能被遗漏。条件不能 fall through 到其他分支。
