# 赋值

赋值具有以下语法：
`(const|var) identifier[: type] = value`.

- `const` 表示 `identifier` 是一个存储不可变值的**常量**。
- `var` 表示 `identifier` 是一个存储可变值的**变量**。
- `: type` 是 `identifier` 的类型注解，如果可以推断 `value` 的类型则可以省略。

<CodeBlock language="zig">{Assignment}</CodeBlock>

常量和变量 _必须_ 有一个值。如果无法给出已知值，则只要提供类型注释，就可以使用能转换为任何类型的 [`undefined`](https://ziglang.org/documentation/master/#undefined)。

<CodeBlock language="zig">{Undefined}</CodeBlock>

在可能的情况下，`const` 值优先于 `var` 值。
