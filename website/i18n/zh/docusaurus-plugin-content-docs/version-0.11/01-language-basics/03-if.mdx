# If 表达式

Zig 的 if 语句接受 `bool` 值（即 `true` 或 `false`）。与 C 或 Javascript 不同，没有任何值可以隐式转换为 bool 值。

在这里，我们引入测试。保存以下代码并使用 `zig test file-name.zig` 编译和运行它。我们将使用标准库中的 [`expect`](https://ziglang.org/documentation/master/std/#std;testing.expect)
函数，如果给定值为 `false`，将导致测试失败。当测试失败时，将显示错误和堆栈跟踪。
